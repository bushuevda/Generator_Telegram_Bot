Функция создает наблюдателя за сообщениями пользователя.

При совпадении ключевого слова или названия меню с сообщением пользователя (в нижних регистрах) выводит файл с текстом.

Аргументы
name_menu- название меню.
key_words- множество ключевых слов.
file_path- путь к файлу.
keyboard- список кнопок меню.
caption- описание файла.

Переменные:

  1. formed_file - подготовленный файл.
  2. formed_keyboard - сформированное меню.
  3. check_user - результат проверки пользователя (False, True).

Условие:

  1. Если пользователь в черном списке или не зарегистрирован вывод соответствующего сообщения.
